# Contributing

Thank you for your interest in contributing to `task-tracker`.

## Development setup

```bash
git clone https://github.com/metyatech/task-tracker.git
cd task-tracker
npm install
```

## Submitting changes

1. Fork the repository and create a feature branch.
2. Add or update tests for any changed behavior.
3. Open a pull request with a clear description of the change.
4. Ensure `npm run verify` passes before submitting.

## Scope

This tool covers **persistent task lifecycle tracking for AI agent sessions**. Please keep PRs scoped to task management and persistence concerns.
